# Тестовое задания для компании "Roistat"

Таблица с вложенностью и добавлением нового пользователя, данные сохраняются в localstorage.

Стэк:
- Vue 2;
- Scss;
- Vue cli;
- Vuex;

Запуск проекта: 
```
npm install
```
```
npm run dev
```
